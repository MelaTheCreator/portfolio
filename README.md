# Frontend Portfolio – Melanie Heß

Personal frontend portfolio by a career changer with a background in education.  
Built with React & Tailwind CSS, focusing on clarity, accessibility, and clean frontend architecture.

---

## About this project

This portfolio showcases my journey into web development and presents selected projects
that reflect my focus on structured, user-friendly, and accessible web interfaces.

Coming from a background in education, I place strong emphasis on clarity, usability,
and well-organized code.

---

## Tech Stack

- **React**
- **Tailwind CSS**
- **React Router**
- **Context API**
- **JavaScript (ES6+)**
- **Vite**

---

## Features

- Light / Dark mode toggle
- Language toggle (German / English)
- Clean, component-based architecture
- Responsive layout
- Separated content and UI logic

---

## Getting Started

Clone the repository and install dependencies:

```bash
npm install
npm run dev
```
